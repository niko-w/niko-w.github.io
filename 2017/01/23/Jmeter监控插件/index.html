  2. 在要监控的服务器上安装插件ServerAgent，解压ServerAgent-2.2.1.zip文件，进入文件下，运行./startAgent.sh（linux下）

Plugin Type Classes
插件提供解析下面这些数据的对象：

  1. AggregateReport = JMeter’s native Aggregate Report, can be saved only as CSV 聚合报告
  2. SynthesisReport = mix between JMeter’s native Summary Report and Aggregate Report, can be saved only as CSV
  3. ThreadsStateOverTime = Active Threads Over Time 线程数随时间的变化曲线
  4. BytesThroughputOverTime 流量随时间的变化曲线
  5. HitsPerSecond 点击率随时间的变化曲线
  6. LatenciesOverTime 延迟随时间的变化曲线
  7. PerfMon = PerfMon Metrics Collector 服务器参数随时间变化曲线注意这个需要在jmx脚本中添加，服务器监控，要指定ip、端口、监控项，还需要指定perfMon的输出文件，或者在命令行指定，如sh jmeter.sh -n -t HTTP请求-baidu.jmx -l 2.jtp -JforcePerfmonFile=true,这样会输出在当前路径。
  8. ResponseCodesPerSecond 响应码随时间变化曲线
  9. ResponseTimesDistribution 响应时间分布
  10. ResponseTimesOverTime 响应时间随时间变化曲线
  11. ResponseTimesPercentiles 响应时间的百分比
  12. ThroughputVsThreads 吞吐率随线程变化曲线
  13. TimesVsThreads = Response Times VS Threads 响应时间随线程变化曲线
  14. TransactionsPerSecond 事务随时间变化曲线
  15. PageDataExtractorOverTime
